What is reQual?

  • reQual is (yet another) FOSS application for qualitative research powered by R programming language

Define reQual

Why R?

  • de facto computational standard in social sciences
    • easier to collaborate with computer-savvy colleagues
  • well-equipped to run statistical evaluations on the backend
  • large ecosystem of scholarly packages
  • Shiny framework in R makes it (relatively) easy to build UI and offload some functionality to the browser

Why reQual?

  • re stands for reproducibility
    • the ideal of computational reproducibility is to connect raw data to final publication by source code and execute the entire pipeline with one-click
      • reproducibility in this sense is unattainable in QDA
  • reQual as a piece of software is a tool to assist in qualitative research
  • reQual as a project is a way to think and experiment with the idea of reproducibility in QDA

Explain reQual’s epistemological motivation.

Can you use reQual?

  • for students to practice basic coding
  • contribute to the development
    • if you are feeling experimental (bug reports, feature discussions and requests)
  • for real-world projects
